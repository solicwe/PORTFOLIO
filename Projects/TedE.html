Project Overview

TedE is a grade tracking app I’ve been working on since 2022. It helps students calculate their GPA easily and see how they improve over time. I also added a fun "Leaderboard" feature so students can see their rankings and compete with friends anonymously.

Key Features

  • Easy GPA Calculator: Just type in your grades, and it calculates the result instantly.
  • Ranking System: A live leaderboard to see who’s top of the class (compete anonymously).
  • Progress Charts: Simple graphs that show if your grades are going up or down each term.
  • Works Everywhere: Designed to look great on both phones and computers.

Technical Implementation

  • Logic Implementation: Wrote the algorithms for weighted grade calculations and ranking logic.
  • UI/UX Design: Created a clean, student-focused interface that simplifies complex data entry.
  • Feature Iteration: Continuously added features based on real user feedback (e.g., the ranking system).

My Contributions

  • Frontend Developer: I was in charge of the website's design and code using React.js.
  • User Interface: I built the screens and menus to be super simple for students to use on both phone and computer.
  • Team Integration: I connected my frontend code with the backend system to ensure the grades and rankings display correctly.

Learning Outcomes

  • Building Websites: Learned how to turn a design idea into a real, working website.
  • Connecting Data: Learned how to link my frontend screens to the database so real information shows up.
  • Team Collaboration: Learned how to work closely with backend developers to make the whole system come together.
