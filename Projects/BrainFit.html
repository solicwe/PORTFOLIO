Project Overview

BrainFit is an innovative assistive cognitive care application specifically designed to support individuals living with Alzheimer's disease. The application provides a suite of interactive activities and engaging games tailored to enhance and maintain cognitive functions in daily life.

Key Features

  • Interactive Cognitive Games: Memory exercises, pattern recognition, and problem-solving activities
  • User-Friendly Interface: Designed specifically for elderly users with large buttons and clear navigation
  • Progress Tracking: Monitor cognitive performance over time with visual analytics
  • Adaptive Difficulty: Activities adjust based on user performance
  • Accessibility Features: High contrast modes, text-to-speech, and simplified interactions

My Contributions

  • Developed responsive UI components using ReactJS with focus on accessibility
  • Designed and implemented user-friendly interfaces optimized for elderly users
  • Collaborated with healthcare professionals to ensure medical accuracy
  • Conducted usability testing with target demographic
  • Optimized application performance for smooth user experience
  • Integrated feedback mechanisms for continuous improvement
